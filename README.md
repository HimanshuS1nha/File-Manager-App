## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Prebuild

   ```bash
   npx expo prebuild
   ```

3. Run the app

   ```bash
   npx expo run:android -> For Android
   npx expo run:ios -> For iOS
   ```

## Features

- View, rename, delete, move and copy files
- Create and delete folders
- Create zip files, delete zip files and zip and unzip files
- Install apk files

## TECH STACK USED

- React Native Expo
- Tailwind CSS
- React native fs

- Zod
